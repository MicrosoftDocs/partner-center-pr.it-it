# <a name="registration-id-number-information"></a>Informazioni sul numero ID registrazione

**Ruoli appropriati**

- Amministratore globale
 
Se si è un provider indiretto, un partner con fatturazione diretta o un rivenditore indiretto e si collabora con clienti nuovi o esistenti nei paesi seguenti, è necessario fornire i numeri ID di registrazione per l'azienda. Se il paese in cui si sta effettuando la società non è elencato di seguito, l'ID registrazione è facoltativo.

### <a name="mandatory-or-optional"></a>Obbligatorio o facoltativo
 
Per i paesi in cui il numero di registrazione è obbligatorio, l'etichetta sopra la casella di testo indica il tipo di numero richiesto.
In questo esempio viene chiesto il numero di registrazione DEL CNPJ (Brasile).

:::image type="content" source="images/registration.png" alt-text="Acquisizione schermo della casella di registrazione":::

Se il numero di registrazione è facoltativo, immettere l'ID di registrazione della società. Non immettere un ID personale.
